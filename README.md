# SMS Scam Detector AI

Machine learning model that classifies SMS messages as "scam" or "safe" using Natural Language Processing.

## 🔥 Features
- Text classification using scikit-learn
- Detects phishing, lottery scam, and urgent-action messages  
- Real-time prediction via command-line interface
- Lightweight: runs on mobile and desktop

## 🛠️ Tech Stack
- **Python 3.13**
- **scikit-learn 1.6.1** - CountVectorizer + Multinomial Naive Bayes
- **NumPy** for numerical operations

## 🚀 How to Run
```bash
pip install scikit-learn
python scam_detector.py
